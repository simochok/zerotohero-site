# Plateformes pour vendre sans site web

## 1. Gumroad
- **URL**: gumroad.com
- **Frais**: Gratuit (prise % sur ventes)
- **Type**: Digital products (services aussi!)
- **Action**: Créer un compte

## 2. Whop
- **URL**: whop.com
- **Frais**: Gratuit pour commencer
- **Type**: Services et produits digitaux

## 3. Payhip
- **URL**: payhip.com
- **Frais**: Gratuit
- **Type**: E-books, services

## Action
Te demander de créer compte Gumroad/Whop pour que je puisse configurer les produits
